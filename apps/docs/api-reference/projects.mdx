# Projects API

The Projects API allows you to organize documents and comparisons into client-specific projects for better workflow management.

## Create Project

Create a new client project to organize documents and comparisons.

<CodeGroup>
```bash cURL
curl -X POST https://api.benee.com/projects \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Acme Corp 2024 Benefits Review",
    "clientName": "Acme Corporation",
    "description": "Annual benefits package review and optimization",
    "renewalDate": "2024-12-31"
  }'
```

```javascript JavaScript
const response = await fetch('/api/projects', {
  method: 'POST',
  headers: {
    'Authorization': `Bearer ${token}`,
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    name: 'Acme Corp 2024 Benefits Review',
    clientName: 'Acme Corporation',
    description: 'Annual benefits package review and optimization',
    renewalDate: '2024-12-31'
  }),
});
```
</CodeGroup>

### Parameters

<ParamField body="name" type="string" required>
  Project name for internal reference
</ParamField>

<ParamField body="clientName" type="string" required>
  Client or organization name
</ParamField>

<ParamField body="description" type="string">
  Project description and objectives
</ParamField>

<ParamField body="renewalDate" type="string">
  Benefits renewal or decision deadline (ISO 8601 format)
</ParamField>

<ParamField body="budget" type="number">
  Project budget or premium target
</ParamField>

<ParamField body="tags" type="array">
  Tags for project categorization
</ParamField>

<ParamField body="metadata" type="object">
  Additional project metadata
</ParamField>

### Response

<ResponseExample>
```json Success Response
{
  "success": true,
  "data": {
    "projectId": "proj_12345",
    "name": "Acme Corp 2024 Benefits Review",
    "clientName": "Acme Corporation",
    "description": "Annual benefits package review and optimization",
    "renewalDate": "2024-12-31T00:00:00Z",
    "status": "active",
    "documentCount": 0,
    "comparisonCount": 0,
    "createdAt": "2024-01-15T10:00:00Z",
    "updatedAt": "2024-01-15T10:00:00Z"
  }
}
```
</ResponseExample>

## List Projects

Retrieve a list of projects with filtering and pagination.

<CodeGroup>
```bash cURL
curl -X GET "https://api.benee.com/projects?status=active&limit=10" \
  -H "Authorization: Bearer <token>"
```

```javascript JavaScript
const params = new URLSearchParams({
  status: 'active',
  limit: '10',
  sortBy: 'updatedAt',
  sortOrder: 'desc'
});

const response = await fetch(`/api/projects?${params}`, {
  headers: {
    'Authorization': `Bearer ${token}`,
  },
});
```
</CodeGroup>

### Query Parameters

<ParamField query="status" type="string">
  Filter by project status: `active`, `completed`, `archived`, `paused`
</ParamField>

<ParamField query="clientName" type="string">
  Filter by client name (partial match)
</ParamField>

<ParamField query="tags" type="string">
  Filter by tags (comma-separated)
</ParamField>

<ParamField query="renewalBefore" type="string">
  Filter projects with renewal dates before this date
</ParamField>

<ParamField query="renewalAfter" type="string">
  Filter projects with renewal dates after this date
</ParamField>

<ParamField query="sortBy" type="string" default="updatedAt">
  Sort by: `name`, `clientName`, `createdAt`, `updatedAt`, `renewalDate`
</ParamField>

<ParamField query="sortOrder" type="string" default="desc">
  Sort order: `asc`, `desc`
</ParamField>

<ParamField query="limit" type="number" default="20">
  Number of results per page (max 100)
</ParamField>

<ParamField query="offset" type="number" default="0">
  Number of results to skip
</ParamField>

  method: 'POST',

    'Authorization': `Bearer ${token}`,

<ParamField body="documentIds" type="array" required>

## Export Project

Export project data and results in various formats.

<CodeGroup>
```bash cURL
curl -X POST https://api.benee.com/projects/proj_12345/export \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{
    "format": "pdf",
    "includeDocuments": true,
    "includeComparisons": true,
    "template": "comprehensive_report"
  }'
```

```javascript JavaScript
const response = await fetch(`/api/projects/${projectId}/export`, {
  method: 'POST',
  headers: {
    'Authorization': `Bearer ${token}`,
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    format: 'pdf',
    includeDocuments: true,
    includeComparisons: true,
    template: 'comprehensive_report'
  }),
});
```
</CodeGroup>

### Parameters

<ParamField body="format" type="string" required>
  Export format: `pdf`, `excel`, `zip`, `json`
</ParamField>

<ParamField body="template" type="string" required>
  Report template: `summary`, `comprehensive_report`, `executive_summary`
</ParamField>

<ParamField body="includeDocuments" type="boolean" default="true">
  Include original documents in export
</ParamField>

<ParamField body="includeComparisons" type="boolean" default="true">
  Include comparison results in export
</ParamField>

<ParamField body="includeAnalytics" type="boolean" default="true">
  Include analytics and insights
</ParamField>

## Archive Project

Archive a completed project to reduce clutter while preserving data.

<CodeGroup>
```bash cURL
curl -X POST https://api.benee.com/projects/proj_12345/archive \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{
    "reason": "Project completed successfully",
    "preserveData": true
  }'
```

```javascript JavaScript
const response = await fetch(`/api/projects/${projectId}/archive`, {
  method: 'POST',
  headers: {
    'Authorization': `Bearer ${token}`,
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    reason: 'Project completed successfully',
    preserveData: true
  }),
});
```
</CodeGroup>

### Parameters

<ParamField body="reason" type="string">
  Reason for archiving the project
</ParamField>

<ParamField body="preserveData" type="boolean" default="true">
  Whether to preserve all project data
</ParamField>

## Delete Project

Permanently delete a project and all associated data.

<CodeGroup>
```bash cURL
curl -X DELETE https://api.benee.com/projects/proj_12345 \
  -H "Authorization: Bearer <token>"
```

```javascript JavaScript
const response = await fetch(`/api/projects/${projectId}`, {
  method: 'DELETE',
  headers: {
    'Authorization': `Bearer ${token}`,
  },
});
```
</CodeGroup>

<Warning>
This action permanently deletes the project and cannot be undone. All associated documents and comparisons will also be deleted.
</Warning>
