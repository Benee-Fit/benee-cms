# Benefits Blueprint

Benefits Blueprint is a powerful feature designed for benefits brokers and insurance consultants to efficiently compare multiple insurance carriers' offerings using AI-powered document processing.

## Overview

The Benefits Blueprint workflow consists of three main stages:

1. **Document Upload**: Upload insurance quote PDFs with intelligent categorization
2. **AI Processing**: Automated data extraction and structuring using advanced AI models
3. **Analysis & Comparison**: Comprehensive market comparison and report generation

## Document Upload System

### Supported File Types
- PDF documents (primary format for insurance quotes)
- Multiple documents per project
- Batch upload capabilities

## AI-Powered Document Processing

### Text Extraction
- Utilizes PDF.co for reliable text extraction from complex insurance documents
- Handles various document formats and layouts
- Asynchronous processing for large documents

### Data Structuring
- Google Gemini AI (gemini-2.5-pro-preview) processes extracted text
- Structured JSON output with standardized schema
- Intelligent recognition of insurance terminology and structures

## Best Practices

### Document Preparation
1. Ensure PDF documents are text-readable (not scanned images)
2. Use clear, standard insurance quote formats
3. Include all relevant plan information in uploads

### Data Validation
1. Review AI-extracted data for accuracy
2. Verify premium calculations match source documents
3. Check coverage details against original quotes

### Comparison Analysis
1. Use consistent time periods for fair comparisons
2. Consider total cost of ownership, not just premiums
3. Factor in benefit differences when analyzing costs

## Troubleshooting

### Common Issues
- **Processing Errors**: Check document format and readability
- **Missing Data**: Verify all required information is present in source documents
- **Comparison Discrepancies**: Review data extraction accuracy
