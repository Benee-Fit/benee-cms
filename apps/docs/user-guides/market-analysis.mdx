# Market Analysis Guide

Benee-CMS provides powerful market analysis tools to help benefits brokers and insurance consultants make data-driven decisions for their clients.

## Analysis Types

### Full Market Comparison

Compare all available carriers across all coverage types to identify the best overall value proposition.

<Steps>
  <Step title="Select Analysis Type">
    Navigate to your project and click **"Create Comparison"** â†’ **"Full Market Analysis"**
  </Step>
  <Step title="Choose Documents">
    Select all relevant quote documents from different carriers
    - Minimum 2 carriers required
    - Include all coverage types for comprehensive analysis
  </Step>
  <Step title="Configure Settings">
    - Set analysis preferences (cost vs. benefits weighting)
    - Choose display options (summary vs. detailed view)
    - Select relevant coverage filters
  </Step>
  <Step title="Generate Analysis">
    Click **"Generate Analysis"** to start the AI-powered comparison process
  </Step>
</Steps>

### API Integration

For advanced users, integrate market analysis capabilities into your existing workflow systems.

<CodeGroup>
```javascript Trigger Analysis
const analysis = await fetch('/api/comparisons', {
  method: 'POST',
  headers: {
    'Authorization': `Bearer ${token}`,
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    name: 'Q1 Market Analysis',
    documentIds: ['doc_1', 'doc_2', 'doc_3'],
    comparisonType: 'full_market'
  }),
});
```

```javascript Get Results
const results = await fetch(`/api/comparisons/${comparisonId}`, {
  headers: {
```

## Troubleshooting

### Common Analysis Issues

<AccordionGroup>
  <Accordion icon="x-circle" title="Incomplete Analysis">
    **Symptoms**:
    - Missing carrier data
    - Partial coverage information
    - Inconsistent results
    
    **Solutions**:
    - Verify document quality and completeness
    - Check for unsupported document formats
    - Ensure all coverage types are represented
    - Contact support for complex document structures
  </Accordion>
  <Accordion icon="clock" title="Slow Processing">
    **Symptoms**:
    - Extended analysis generation time
    - Timeout errors during processing
    
    **Solutions**:
    - Reduce number of documents per analysis
    - Ensure documents are optimally formatted
    - Check system status for service issues
    - Retry analysis during off-peak hours
  </Accordion>
</AccordionGroup>
